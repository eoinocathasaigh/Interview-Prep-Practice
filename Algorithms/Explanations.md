<h1>Algorithms</h1>
Algorithms are basically a set of rules/instructions followed by a program to solve a certain problem. They are the heart of programming and are used to manipulate data structures, perform calculations, and solve complex problems efficiently. Common examples of such you may have heard of include:
- Sorting algorithms (e.g., Quick Sort, Merge Sort, Bubble Sort)
- Searching algorithms (e.g., Binary Search, Linear Search)
- Graph algorithms (e.g., Dijkstra's Algorithm, Depth-First Search, Breadth-First Search)

The following file will explore these algorithms in detail, providing explanations as to what they are, how they work and when/where to use them. It will also include code snippets in Java to demonstrate their implementation. I will try my best to update this with some examples from actual interview stuff as well, so stay tuned!

<h2>Sorting Algorithms</h2>
Sorting algorithms are used to arrange data in a specific order (ascending or descending). They are fundamental for optimizing the efficiency of other algorithms (like search algorithms) and for organizing data.
